Installing MySQL for qiki Unit Tests

  • Download MySQL Server . You will have to log in to Oracle. You may choose either the installer, or the full download. If you choose the installer, be sure to add the Python connector for your version of Python.
  • pip install six
  • If you didn't already, install the MySQL Python Connector
    pip install %1 --allow-external mysql-connector-python mysql-connector-python
    Thanks to Geert JM Vanderkelen for these instructions.
  • mysql --port NNNNNN -uroot -p

    GRANT CREATE,INSERT,SELECT,DROP ON unittest_database.* TO 'unittest-bot'@localhost;
